Промышленное металлообрабатывающее оборудование может быть многофункциональным, с возможностью выполнения нескольких функций, или узкоспециализированным и имеет четкую классификацию с учетом типа выполняемого станком функционала.

Ассортимент для выполнения заготовительных работ на сайте представлен гильотинными ножницами, предназначенными для резки крупных металлических листов, установками лазерной и плазменной резки для термического раскроя, ленточнопильным оборудованием для обработки различных заготовок (труб, профилей, двутавровых балок).
