Коуши типа КД

Данное оборудование заказывается для различных шахт по заявленным клиентом техническим характеристикам. Наши заказчики должны заполнить опросный лист по определённой форме. Этот лист подписывается директором предприятия-заказчика и заверяется печатью. Мы рассчитаем для каждого заказчика стоимость продукции и предоставим индивидуальные предложения. Коуши незаменимы для такелажных работ. Они предохраняют стальной трос от перетирания. Изготавливаем их из высокоуглеродистой стали. После изготовления они обязательно оцинковываются, чтобы избежать коррозии металла.
